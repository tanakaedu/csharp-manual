## 複合代入演算子

|処理                   |複合代入演算子|例|算術演算に書き換えると|
|:---------------------:|:----------|:--|:-----------------|
|aに5を加える           |            |    |a = a + 5;         |
|aから5を引く           |            |    |a = a - 5;         |
|aを5倍する             |            |   |a = a * 5;         |
|aを5で割る             |            |   |a = a / 5;         |
|aを5で割った余りにする   |            |   |a = a % 5;         |
